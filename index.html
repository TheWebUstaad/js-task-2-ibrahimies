📘 JS Concepts Explained

1. input Event

Jab user kisi input ya textarea me likhta hai to input event turant trigger hota hai. Iska use live changes detect karne ke liye hota hai jaise word count, live validation, etc.

2. split(" ")

Ye method string ko tod kar array me convert karta hai. Agar space pass karein to har word alag ho jata hai. Useful for counting words.

3. document.createElement('div')

JS se naya HTML element banane ke liye use hota hai. Isse DOM me naye boxes, buttons, etc. dynamically add kiye ja sakte hain.

4. Math.random() + Math.floor()

Math.random() 0 aur 1 ke beech random number deta hai. Math.floor() decimal ko hata kar nearest lower integer banata hai. Dono mil ke array me random index choose karne me use hote hain.

5. nextElementSibling

HTML me kisi element ke baad wala agla sibling element ko access karta hai. FAQ toggle me useful hota hai answer div ko access karne ke liye.

6. event.target

Jis element par click ya event hua hota hai, wo element ko target karta hai. Isse us element ki properties access ki ja sakti hain.

7. data-attributes

Custom attributes jaise data-emoji store karte hain additional info. dataset.emoji se uski value access kar sakte ho.

8. classList.toggle()

CSS class ko add/remove karne ka shortcut hai. Agar class lagi ho to hata dega, nahi lagi ho to laga dega. Toggle effects ke liye best.

9. filter()

Array ke items ko condition ke basis par filter karta hai. Jaise: fruit ka naam input me likhe huye text se match ho raha hai ya nahi.

10. includes()

Ye check karta hai ke kisi string me ek choti string exist karti hai ya nahi. Case-sensitive hota hai. Useful for search filtering.
